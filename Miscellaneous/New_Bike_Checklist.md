# ✅ New Bike Checklist

↖️ [Return to the main file](../README.md)

### Basics
- [ ] Inspect frame for damage
- [ ] Check tire pressure
- [ ] Ensure brakes are working properly
- [ ] Adjust saddle and handlebars

### Accessories
- [ ] Helmet
- [ ] Lights and reflectors
- [ ] Lock and bell
- [ ] Water bottle and cage

### Documentation
- [ ] Read user manual
- [ ] Register warranty if available
- [ ] Note serial number
