# ✅ Skincare Checklist

↖️ [Return to the main file](../README.md)

### Morning Routine
- [ ] Cleanser
- [ ] Toner or hydrating mist
- [ ] Serum (vitamin C, hyaluronic acid)
- [ ] Moisturizer
- [ ] Sunscreen

### Evening Routine
- [ ] Makeup remover (if needed)
- [ ] Cleanser
- [ ] Exfoliant (2–3x a week)
- [ ] Serum (retinol, niacinamide, etc.)
- [ ] Night cream or mask

### Weekly Care
- [ ] Clay or sheet mask
- [ ] Lip scrub and balm
- [ ] Body lotion/oil
