# ✅ Photos Cleanup Checklist

↖️ [Return to the main file](../README.md)

### Digital Photos
- [ ] Sort by date and event
- [ ] Delete duplicates with a cleanup tool
- [ ] Rename files consistently
- [ ] Organize into folders or albums
- [ ] Backup to cloud & external storage

### Printed Photos
- [ ] Gather all prints in one place
- [ ] Remove damaged or duplicate prints
- [ ] Store in albums or photo boxes
- [ ] Label by year/event
- [ ] Consider scanning for digital copy
