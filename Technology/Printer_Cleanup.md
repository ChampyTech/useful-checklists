# ✅ Printer Cleanup Checklist

↖️ [Return to the main file](../README.md)

### Physical Cleaning

- [ ] Turn off and unplug the printer
- [ ] Remove paper from the tray
- [ ] Wipe exterior surfaces with a soft cloth
- [ ] Clean the paper path with compressed air

### Internal Maintenance

- [ ] Check and clean rollers if accessible
- [ ] Remove any paper jams or debris
- [ ] Inspect and clean the print head area
- [ ] Check for toner or ink spills

### Software and Settings

- [ ] Remove unused printers from device list
- [ ] Clear print queue
- [ ] Delete old print jobs
- [ ] Update printer drivers if needed

### Testing

- [ ] Print a test page
- [ ] Check for quality issues
- [ ] Verify color accuracy
- [ ] Confirm all features are working
