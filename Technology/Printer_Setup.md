# ✅ Printer Setup Checklist

↖️ [Return to the main file](../README.md)

### Unboxing and Placement

- [ ] Unbox the printer carefully
- [ ] Remove all protective materials
- [ ] Place on stable, flat surface
- [ ] Ensure proper ventilation

### Initial Configuration

- [ ] Load paper into the tray
- [ ] Install ink cartridges or toner
- [ ] Connect power cable
- [ ] Turn on the printer

### Connection Setup

- [ ] Choose connection method (Wi-Fi, USB, Ethernet)
- [ ] Download and install drivers from manufacturer
- [ ] Add printer to your devices
- [ ] Test the connection

### First Use

- [ ] Print a test page
- [ ] Adjust print quality settings
- [ ] Configure default preferences
- [ ] Set up mobile printing if available
