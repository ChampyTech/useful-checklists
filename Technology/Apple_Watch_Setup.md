# ✅ Apple Watch Setup Checklist

↖️ [Return to the main file](../README.md)

### Initial Setup

- [ ] Unbox and inspect the device
- [ ] Charge the Apple Watch
- [ ] Place watch on wrist and fasten band
- [ ] Open Watch app on your iPhone

### Pairing and Configuration

- [ ] Hold iPhone near Apple Watch to pair
- [ ] Sign in with Apple ID
- [ ] Enable location services if desired
- [ ] Set up payment method (Apple Pay)

### Customization

- [ ] Choose and customize watch face
- [ ] Add complications (widgets)
- [ ] Adjust display brightness
- [ ] Set notification preferences

### Apps and Features

- [ ] Download essential apps
- [ ] Enable health and fitness tracking
- [ ] Set up emergency contacts
- [ ] Test Siri and voice commands
