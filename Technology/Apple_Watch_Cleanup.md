# ✅ Apple Watch Cleanup Checklist

↖️ [Return to the main file](../README.md)

### Physical Cleaning

- [ ] Wipe the screen with a soft, lint-free cloth
- [ ] Clean the band and back of the device
- [ ] Remove any dust from crevices
- [ ] Sanitize if needed (avoid excess moisture)

### Software Cleanup

- [ ] Review and close unused apps
- [ ] Clear notifications and alerts
- [ ] Delete old messages and emails
- [ ] Remove unnecessary watch faces

### Storage and Performance

- [ ] Check available storage space
- [ ] Delete old photos and videos
- [ ] Clear app cache if possible
- [ ] Restart the watch for a fresh start

### Maintenance

- [ ] Update to the latest watchOS version
- [ ] Review battery usage
- [ ] Unpair and re-pair if experiencing issues
- [ ] Check for loose bands or damage
