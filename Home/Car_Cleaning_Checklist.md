# ✅ Car Cleaning Checklist

↖️ [Return to the main file](../README.md)

### Exterior

- [ ] Wash the body and windows
- [ ] Clean the wheels and tires
- [ ] Wipe down mirrors and lights
- [ ] Remove dirt from door seals

### Interior

- [ ] Vacuum seats and floor
- [ ] Wipe dashboard and steering wheel
- [ ] Clean windows from inside
- [ ] Empty and wipe storage compartments

### Final Touches

- [ ] Apply tire shine if desired
- [ ] Vacuum trunk area
- [ ] Wipe down door handles
- [ ] Check for any missed spots
