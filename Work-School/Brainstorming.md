# ✅ Brainstorming Checklist

↖️ [Return to the main file](../README.md)

### Preparation

- [ ] Define the problem or topic clearly
- [ ] Set a time limit for the session
- [ ] Gather relevant materials and references
- [ ] Create a comfortable meeting space

### During Session

- [ ] Encourage all ideas without judgment
- [ ] Write down all suggestions
- [ ] Build on others' ideas
- [ ] Avoid criticism or negativity

### Organization

- [ ] Group related ideas together
- [ ] Identify patterns and themes
- [ ] Prioritize the best ideas
- [ ] Combine similar concepts

### Follow-Up

- [ ] Document the final list
- [ ] Share results with participants
- [ ] Assign action items
- [ ] Schedule implementation review
